# Before Cloud: What Problems Did We Have?

Before cloud computing, companies used on-premises datacenters, and it came with huge drawbacks:

* High Upfront Cost

* Scalability Issues

* Underutilization

* Maintenance Overhead

* Limited Global Reach

* Slow Deployment
